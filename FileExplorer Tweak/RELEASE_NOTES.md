# FileExplorer Tweak v1.0.0

Initial release of the FileExplorer Tweak utility.

## Highlights
- Enable the `FolderType` registry value (`NotSpecified`) for a File Explorer performance tweak.
- One-click buttons to apply or remove the tweak, with automatic Explorer restart.
- Built-in backup to `.reg` and restore from backup.

## Notes
- This tool modifies a per-user registry key under `HKCU`.
- A .NET runtime is required if you are not using the self-contained build.
- This file has been scanned by Bitdefender AV as clean.
